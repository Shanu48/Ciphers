Vigenere Cipher

The Vigenère Cipher is a classical encryption technique that uses a keyword to shift the letters of the plaintext message, creating a polyalphabetic cipher. This method provides a higher level of security compared to simple substitution ciphers by employing multiple Caesar ciphers based on the letters of the keyword.
Encryption - In a Vigenère cipher, each letter of the plaintext is shifted along the alphabet by the number of positions indicated by the corresponding letter of the keyword.
1) The keyword is repeated so that it matches the length of the plaintext.
2) Each letter of the plaintext is shifted according to the corresponding letter of the keyword. For example, if the keyword is "LEMON", and the plaintext is "ATTACKATDAWN", the keyword is repeated to "LEMONLEMONLE".
3) The first letter of the plaintext 'A' is shifted by 'L' (11 positions) resulting in 'L', the second letter 'T' is shifted by 'E' (4 positions) resulting in 'X', and so on. The resulting ciphertext is "LXFOPVEFRNHR".
Decryption - To decipher the message, the recipient must use the same keyword to reverse the shift.
1) The keyword is repeated to match the length of the ciphertext.
2) Each letter of the ciphertext is shifted backward according to the corresponding letter of the keyword. For example, using the same keyword "LEMON" and the ciphertext "LXFOPVEFRNHR", the keyword is repeated to "LEMONLEMONLE".
3) The first letter of the ciphertext 'L' is shifted backward by 'L' (11 positions) resulting in 'A', the second letter 'X' is shifted backward by 'E' (4 positions) resulting in 'T', and so on. The resulting plaintext is "ATTACKATDAWN".
